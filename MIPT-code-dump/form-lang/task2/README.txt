Формат ввода:
В первой строке должны быть записаны начальный нетерминал (заглавная латинская буква) и, через пробел, слово, которое либо состоит только из строчных латинских букв, либо равно "-" без кавычек. Последнее означает пустое слово.
В каждой из последующих строк должны быть записаны правила в следующем формате: начальный нетерминал (заглавная латинская буква) и, через пробел, правая часть правила, которая либо состоит только из строчных и заглавных латинских букв, либо равна "-" без кавычек. Последнее означает пустое слово.

Формат вывода:
Программа выводит "YES", если грамматика допускает слово, и "NO" иначе.
